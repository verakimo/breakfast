# Hyvä aamiainen -sovellus
* Sovelluksessa käyttäjät voivat jakaa aamiaisreseptejä. Resepti sisältää tarvittavat ainekset, valmistusohjeen ja valmistusajan.
* Käyttäjä voi luoda tunnuksen sekä kirjautua sisään ja ulos sovelluksesta.
* Käyttäjä voi lisätä uusia reseptejä sekä muokata ja poistaa omia reseptejään.
* Käyttäjä näkee sovellukseen lisätyt reseptit.
* Käyttäjä voi etsiä reseptejä hakusanalla.
* Käyttäjäsivulla näytetään käyttäjän lisäämien reseptien määrä sekä lista käyttäjän lisäämistä resepteistä.
* Käyttäjä voi valita reseptille yhden tai useamman luokittelun. Mahdollisia luokitteluja ovat esimerkiksi:
** Aamiaisen ominaisuus: proteiinipitoinen, hiilihydraattipitoinen tai juhla-aamiainen
** Aamiaisen tyyppi: puuro, mysli tai jogurtti, munaruoka, voileipä tai muu
** Ruokavalio: laktoositon, gluteeniton tai vegaaninen
* Käyttäjä voi kirjoittaa reseptiin kommentin ja antaa sille arvosanan. Reseptin sivulla näytetään käyttäjien kommentit ja reseptin keskimääräinen arvosana.
* Mahdollisuuksien mukaan reseptiin voi lisätä myös tiedot allergeeneista ja ravintosisällöstä sekä kuvan valmiista ruoasta.
Sovelluksen pääasiallinen tietokohde on aamiaisresepti. Toissijainen tietokohde on reseptiin liittyvä kommentti ja arvosana.
